|                       | **Term**                | **Definition**                                                                                                                                                                                                      | **Note** |
|-----------------------|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------|
| **Classes**           |                         |                                                                                                                                                                                                                     |          |
|                       | `HeritageAsset`         | A physical or tangible item, structure, site, or object that holds historical, cultural, architectural, archaeological, or artistic significance and is considered worthy of preservation and protection.           |          |
|                       | `EvaluationDescription` | An activity that consists in evaluating risks relative to some criteria.                                                                                                                                            |          |
|                       | `Risk`                  | The possibility of a loss of value to the heritage asset.                                                                                                                                                           |          |
|                       | `PriorityLevel`         | The level of priority assigned to a risk during an evaluation activity.                                                                                                                                             |          |
|                       | `CriterionSetter`       | An abstraction of something (e.g. a process, a requirement, etc.) that sets a criterion according to some condition.                                                                                                |          |
|                       | `Measure`               | A quantifiable aspect of something that expresses a more precise and standardised evaluation of it, based on assigning numerical values or metrics to some specific attributes.                                     |          |
|                       | `Magnitude`             | The measure representing the overall impact and severity of a risk.                                                                                                                                                 |          |
|                       | `Uncertainty`           | The measure representing the unpredictability associated with the factors involved in evaluating and quantifying a risk.                                                                                            |          |
|                       | `QuantitativeCondition`             | A specific numerical factor that is used as a threshold by a criterion setter during the evaluation of a risk.                                                                                                                |          |
|                       | `AcceptanceLevel`       | An agreed-upon level of risk that determines whether it can be tolerated or not.                                                                                                                                    |          |
| **Object properties** |                         |                                                                                                                                                                                                                     |          |
|                       | `hasAcceptanceLevel`   | A relationship between an evaluation description and an acceptance level.                                                                                                                                           |          |
|                       | `hasPriorityLevel`     | A relationship between an evaluation description and a priority level.                                                                                                                                              |          |
|                       | `setsCriterion`         | A relationship between a criterion setter and a measure.                                                                                                                                                            |          |
|                       | `describes`             | A relationship between an evaluation description and a heritage asset.                                                                                                                                              |          |
|                       | `evaluates`             | A relationship between an evaluation description and a risk.                                                                                                                                                        |          |
|                       | `withCondition`         | A relationship between a criterion setter and a condition.                                                                                                                                                          |          |
|                       | `usesCriterion`         | A relationship between an evaluation description and a criterion setter.                                                                                                                                            |          |
|                       | `onAttribute`         | A relationship between a condition and a quality that is being considered to set that condition.                                                                                                                                            |          |
| **Data properties**   |                         |                                                                                                                                                                                                                     |          |
|                       | `hasDegreeOfConfidence` | A property that assigns a quantitative value to an uncertainty measure.                                                                                                                                             |          |
|                       | `hasProbableEstimate`   | A property that assigns a estimate value for the most likely case scenario to a measure.                                                                                                                            |          |
|                       | `hasQuantitativeConditionValue`     | A property that assigns a value to a quantitative condition.                                                                                                                                                                     |          |
| **Individuals**       |                         |                                                                                                                                                                                                                     |          |
|                       | `treat-asap`            | An acceptance level that expresses the need to treat the risk as soon as possible.                                                                                                                                  |          |
|                       | `review-asap`           | An acceptance level that expresses the need to review the risk as soon as possible.                                                                                                                                 |          |
|                       | `review-later`          | An acceptance level that expresses the possibility to review the risk in a later time.                                                                                                                              |          |
|                       | `no-action`             | An acceptance level that expresses the possibility to withold treatment for now.                                                                                                                                    |          |
|                       | `catastrophic`          | A priority level according to which all or most of the heritage asset value is likely to be lost in a few years.                                                                                                    |          |
|                       | `extreme`               | A priority level according to which significant damage to all the heritage asset, or total loss of a significant fraction of the heritage asset, is possible in approximately one decade.                           |          |
|                       | `high`                  | A priority level according to which significant loss of value to a small fraction of the heritage asset, or a small loss of value in most or a significant fraction of the heritage asset is likely in one century. |          |
|                       | `medium`                | A priority level according to which small damage or loss of value to the heritage asset is likely over many centuries.                                                                                              |          |
|                       | `low`                   | A priority level according to which minimal or insignificant damage or loss of value to the heritage asset is likely over many millennia.                                                                           |          |
